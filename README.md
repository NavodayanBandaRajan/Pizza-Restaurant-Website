# 🍕 Lapinoz Pizza House Website

A modern and responsive pizza restaurant website built using HTML, CSS, and JavaScript.

## ✨ Features

- Fixed Navigation Bar
- Smooth Scrolling
- Typing Text Animation
- About Section
- Pizza Menu Cards
- Image Gallery with Lightbox
- Online Order Form
- Contact Form
- Footer Section
- Mobile Responsive Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Purpose

This project is created to practice frontend web development and build a real-world restaurant website.
